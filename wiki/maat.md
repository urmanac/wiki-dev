# Maat

A todo list app with tags, filtering, multiple lists and the ability to share your task lists.

**Note:** This app is in alpha, so it may not work as intended.

A grant proposal for this project was accepted by the Urbit Foundation, which you can read here: [urbit.org/grants/maat](https://urbit.org/grants/maat) <br>
If you have an idea, want to learn as an apprentice or fancy getting paid as a developer, please, have a look over the Urbit Foundation's grants program here: [urbit.org/grants](https://urbit.org/grants)
