# Knox

A vault for web2 passwords with a browser extension and easy-to-use client interface. Instead of waiting for the online services we use (email, banking, etc.) to adopt *Log in with Urbit*, we can use Knox. It can generate strong passwords, easily cycle passwords, autofill login forms with the correct credentials, trivially grab an arbitrary password with a command, everything you expect a first class password manager (BitWarden, LastPass) to do, but it lives on your urbit.

**Note:** this app is still in beta.

A grant proposal for this project was accepted by the Urbit Foundation, which you can read here: [urbit.org/grants/knox](https://urbit.org/grants/knox) <br>
If you have an idea, want to learn as an apprentice or fancy getting paid as a developer, please, have a look over the Urbit Foundation's grants program here: [urbit.org/grants](https://urbit.org/grants)
